# 📚 DZ Bot (Homework Assistant)

**Telegram Bot:** [@DZ_K122_bot](https://t.me/DZ_K122_bot)

Этот бот создан для удобного ведения списка домашних заданий. Он позволяет добавлять задания на конкретные даты, просматривать их и удалять. Каждый пользователь имеет свой личный изолированный список.

## ✨ Возможности

- **📝 Добавление домашнего задания:** Пошаговый ввод заданий с привязкой к дате.
- **➕ Продолжить ввод:** Возможность вернуться к любой существующей дате и добавить новые задания в конец списка.
- **✏️ Удаление конкретного задания:** Удаление не всего списка, а только одной конкретной позиции за выбранную дату.
- **📋 Просмотр списка:** Быстрое отображение всех ваших текущих заданий с сортировкой по датам.
- **🗑️ Очистка заданий:** Возможность удалить все задания сразу или только за конкретную дату.
- **🛡️ Приватность:** База данных заданий разделена по пользователям. Каждый видит и редактирует только свои записи.

## 🚀 Как пользоваться

Бот имеет простой и интуитивно понятный интерфейс с кнопками.

### Основной сценарий: Добавление нового задания
1.  Нажмите кнопку **«📝 Добавить ДЗ»**.
2.  Введите дату в формате **ДД.ММ.ГГГГ** (например: `26.02.2026`).
3.  После выбора даты вводите текст заданий по одному. Бот будет сохранять их и запрашивать следующее.
4.  Когда все задания на эту дату будут введены, нажмите кнопку **«⛔ Стоп»**, чтобы завершить ввод и сохранить список.

### 🔄 Продолжить ввод в существующую дату
Если вы хотите добавить ещё задания на дату, которая уже есть в вашем списке:
1.  Нажмите кнопку **«➕ Продолжить ввод»**.
2.  Выберите из списка дату, в которую хотите добавить новые задания (введите её в том же формате).
3.  Бот покажет текущие задания за эту дату.
4.  Вводите новые задания по одному. Они автоматически добавятся **в конец списка** с правильной нумерацией.
5.  Нажмите **«⛔ Стоп»**, когда закончите.

### ✏️ Удаление конкретного задания
Если вы ошиблись или нужно убрать один пункт из уже готового списка:
1.  Нажмите кнопку **«✏️ Удалить задание»**.
2.  Выберите из списка дату, за которую нужно удалить задание (введите её).
3.  Бот покажет все задания за эту дату с номерами.
4.  Введите порядковый номер задания, которое хотите удалить.
5.  Оставшиеся задания автоматически перенумеруются.

### Другие команды и кнопки
*   **`/list`** или **«📋 Показать весь список»** — Показать все ваши задания, сгруппированные по датам.
*   **«🗑️ Очистить»** — Открывает меню очистки:
    *   **«🧹 Очистить всё»** — Полностью удаляет все ваши задания.
    *   **«📅 Удалить по дате»** — Удаляет задания только за конкретную дату.
*   **`/help`** или **«❓ Помощь»** — Показать справочное сообщение с инструкцией.
*   **«⛔ Стоп»** — Отмена текущего действия или завершение ввода.

## 🔒 Важно
Бот построен с учетом конфиденциальности. **Каждый пользователь видит и работает только со своим списком заданий.** Чужие домашние работы недоступны для просмотра или изменения.

## 🛠 Технологии
*   **Язык:** Python
*   **Библиотека:** Aiogram 3.x
*   **База данных:** JSON (хранится в файле `homework_data.json`)
*   **Хранение данных:** Индивидуально для каждого пользователя (ключ — Telegram ID)
